What can we help you with?

  • Where should I begin?
    Try loading one of the examples.
    From the File menu choose Examples.. Try the Simple Robot Example
  • A Specific Block
    The best way to learn more about what a block does it to hover your mouse pointer over it, for most blocks one will see a small bit of hovertext detailing the usage of the block.
  • Deploying Code
    1. Click the Download button in the upper right corner.
    2. Then click the Download! button in the pop-up window.
    3. Uncompress the contents of the zip folder to a new folder
    4. Open Netbeans on your computer. (Don't have it? Follow the tutorial here)
    5. From the Netbeans File menu choose Open Project..
    6. Navigate to where you saved the folder we uncompressed earlier
    7. The icon should be a little rocket
    8. Double click to open the project
    9. Vola! Make sure your robot is connected to your computer and you are ready to press the large green Play Button to start the download to your robot.

    Check out the Getting started with Java from WPI for more information.
  • Copying Blocks
    One can easily copy blocks by right clicking the block they wish to copy, and choosing duplicate.
    NOTE: all blocks that are contained within the block will also be copied.
